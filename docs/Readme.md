# Docs

## Using Sphinx

1. create a new virtual enviorment (venv)

    ```bash
    py -m venv sphinxvenv
    ```

2. install sphinx

    ```bash
    pip install sphinx
    ```

3. create docs directory

    ```bash
    mkdir docs
    ```

4. cd into directory

    ```bash
    cd ./docs
    ```

5. run sphinx quickstart

    ```bash
    sphinx-quickstart
    ```
